SMART HOSPITAL

Project Overview

The Smart Hospital Role-Based Appointment System is a full-stack web application developed using the MERN stack (MongoDB, Express.js, React.js, and Node.js). The system enables patients to book appointments, doctors to manage schedules, and administrators to monitor overall hospital activities. By implementing Role-Based Access Control (RBAC), the system ensures secure access and efficient workflow management. It improves hospital productivity, reduces manual errors, and enhances the patient experience.
